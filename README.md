# 優雅登入系統

響應式的註冊/登入頁面，具備完整表單驗證功能。

## DEMO

[here](https://jeffrey0117.github.io/login-site/)

## 技術棧

- HTML5 + CSS3 + JavaScript
- Bootstrap 5.3
- Font Awesome 6.3

## 主要功能

- ✅ 表單驗證（姓名、信箱、密碼）
- 🔒 密碼強度檢測
- 👁️ 密碼顯示/隱藏切換
- 📱 響應式設計
- � 漸入動畫效果
- 📧 成功提示 Modal

## 使用方式

直接在瀏覽器開啟 `index.html` 即可。

## 預覽

- **桌面版**：左右分欄布局
- **手機版**：垂直堆疊布局
